\# 開発環境構築



\## 必要環境



Node.js  

Git



---



\## クローン



git clone https://github.com/Toshi-Suzu/akw-deepdata.git



cd akw-deepdata



---



\## 依存関係



npm install



---



\## 開発サーバ



npm run dev



---



\## ブラウザ



http://localhost:3000



---



\## 環境変数



.env.local



NEXT\_PUBLIC\_SUPABASE\_URL

NEXT\_PUBLIC\_SUPABASE\_ANON\_KEY

SUPABASE\_SERVICE\_ROLE\_KEY

ADMIN\_TOKEN



を設定する。
